# kakashi (iruka++)
![](./iruka-kakashi.webp)

**Correct** and **performant** data structures and algorithms

kakashi reimplements [iruka](https://github.com/jeffzh4ng/iruka) with a larger
emphasis on correctness (pbt and verification) and performance (vectorization).

### Roadmap
**Data Structures**
  - Sequences: `Seq`, `Heap`, `Vector`, `Matrix`, `Tensor`
  - Graphs: `List`, `Tree`, `Graph`

**Algorithms**

*Big Data (Linear Algebra)*

- Least Squares Regression, Subspace Embeddings, Net Arguments
- Matrix Chernoff, Subsampled  Randomized Hadamard Transform, Approximate Matrix Product, CountSketch
- CountSketch, Affine Embeddings
- Low Rank Approximation, Sketching-based Preconditioning
- Leverage Score Sampling, Distributed Low Rank Approximation
- Distributed Low Rank, L1 Regression
- L1 Regression, Introduction to Streaming
- Estimated Norms in a Data Stream, Heavy Hitter Intuition
- L0 Estimation
